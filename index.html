What was challenging about Assignment 4:

There were several things I thought were challenging completing Assignment 4:

  • Setting up the server.
  • The posts/n page.
  • Serving the CSS, Javascript, and Benny picture.

What I learned:

I found a YoutTube video that helped me set up the server using the HTTP node.js library

It took me a while to figure out how to serve the posts/n page. It took some talking with ChatGPT to learn that I had to use req.url.substring and the isNaN function to test if n was a valid number.

The hardest part of this assignment for me was serving style.css, index.js, and benny.jpg. At first, I didn't realize that I had to set up responses for those files and tried to include them when serving my html files. I was really confused with Google and ChatGPT's answers, and ended up having an aha moment and figuring the solution on my own.
